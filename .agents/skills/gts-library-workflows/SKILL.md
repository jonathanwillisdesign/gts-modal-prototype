# GTS Library Workflows

Use this skill to perform common GTS Central Library workflows. Keep execution aligned with repo conventions.

## Workflow Index
- `add-icon.md`: Add a new SVG icon to the sprite system and update types/story.
- `add-token.md`: Add a new design token following Style Dictionary + DTCG conventions.
- `build-tokens.md`: Build and validate design tokens via Style Dictionary.
- `compare-preprod-header.md`: Compare library Header component against pre-prod retailer header with `agent-browser` (auth + checklist).
- `commit-and-push.md`: Commit staged changes and push using the repo format.
- `convert-color.md`: Convert between hex and OKLCH for token work.
- `create-component.md`: Create a new component following library conventions (files, CVA, Base UI, Storybook).
